# Aldi Campaign Analysis

In 2023, Aldi launched a Disney-themed campaign aimed at attracting more customers. This analysis evaluates the feasibility and potential impact of that campaign.
The project was carried out using Python, SQL, and Power BI. The results are presented in this ![Presentatie_Aldi_Kian](Presentatie_Aldi_Kian.odp). Please note that the underlying data remains confidential.
